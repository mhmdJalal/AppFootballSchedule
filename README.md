# AppFootballSchedule
This is my portpolio project of Android Application. I used kotlin language in this app.

Some features in this app is:
1. Next Match Schedule.
2. Last Match Schedule.
3. Detail Match.
4. Detail Team.
5. Detail Player.
6. Integrated with SQLite.
